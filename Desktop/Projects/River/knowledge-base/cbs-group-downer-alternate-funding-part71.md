### Slide 3
Program Description
CBS’ Partner, Certificate Issuer and Auditors have confirmed:

ApproachThe best approach is to treat the fleets as a series of tranches, defined by the number of sets that may be modified in a given window prior to Winter or Summer
Brings forward the ESC payments (early as possible)
Allows for certificate issue by similarity for sets in the tranche

Reference ModelCertificate issue will require development of a baseline energy reference model for each tranche
Data from each set in the tranche post modification will need to be collected and energy savings determined over a Summer and Winter
Normalisation for passenger loads is permitted

Pilot SetFor the NSW Energy Savings Scheme, the planned pilot set does not provide any useful results or contribution to the verification method.

### Slide 5
Engagement Model
CBS Group proposes the following engagement model:
Monthly Retainer | Engineering support, scheme admin & project management$10k/month over 2 years for 3 tranches
ESC commission15% paid upon receipt of ESC funds

This model provides for an annual cost of $120k to support the project. The remainder of costs, including those of the certificate issuer and auditor incorporated into a commission on the certificates issued, paid upon receipt of funds.
