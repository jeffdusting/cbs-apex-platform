### Slide 8
Basis of Calculations
Key Parameters

SGT Fleet :  41 Sets, 16 HVAC per set

Energy Savings/ HVAC = 13MWh per annum
As calculated based upon models developed by Downer with the support of Sigma, assessed as conservative to avoid ‘over promising to Sydney Trains


Scheme Formulae
ESC = 1.06 x MWh saved x No Years Claimed in advance

ESC Price
Market prices is trending upwards
Min price in last 12 months is $20, with a peak of $31
We have used $27 which is the volume weighted price for the past 12 months
Example: SGT Energy Savings (software mod only)

ESCs	= upfront ESCs (first 7 years) + topup (remaining 3 years)
ESCs (upfront) = 1.06 x MWH/year x #HVAC x years claimed= 1.06 x 13 x 656 x 7= 63,227 ESCs

ESCs (Topup)	= 1.06 x 13 x 656 x 3
	= 27,119

Gross Revenue
ESC Market Price      	= 22/ 27 / 33
Gross Revenue ($m) 	= 1.4 / 1.7 / 2.2 (Upfront)
Gross Revenue ($m)	= 0.6 / 0.7 / 0.9 (Upfront)

Net Revenue (total)	= 1.4 / 1.7 / 2.2

   Context
1. The NSW Energy Savings Scheme (ESS) has the potential to deliver funding back to Downer based on software and hardware
   upgrades to the Sydney train fleets.
   • It requires application of a framework, developed for fixed infrastructure, to mobile rolling stock, a new application of the
     ESS.
   • The application requires:
      • an approach that successfully models an increased number of complex variables
      • satisfying ESS auditors and successfully managing the disposition of certificates on a live ESS trading platform
   • To take advantage of the software upgrade, CBS requires a commencement date of no later than 30 Nov 2020 to be
     certain SGT software modifications are compliant for ESCs.
2. CBS has:
   • identified a path to manage these factors, and retained a qualified ESC (Energy Saving Certificates) issuer as a partner
   • We have proposed a value share model whereby Downer provides the data and pays CBS Group a recoupable retainer
     through project development
   • In response to Sean McCormack's request we have also costed a non-shared risk, T+M engagement
3. Shared risk/shared value is our preferred option.
   • It demonstrably delivers less risk and more value to Downer
   • However this requires executive sponsorship – hence our request for your review today to determine Downer's appetite for
     this proposition.

Program Description
CBS’ ESC Partner, Certificate Issuer and Auditors have confirmed:

•   Approach
    The best approach is a ‘Measurement and Verification’ method treating the
    fleets as a series of tranches, defined by the number of sets that may be
    modified in each window prior to Winter or Summer
    • Brings forward the ESC payments (early as possible)
    • Allows for software and hardware modifications to be ‘assessed’ as
       tranches
    • Downer are the ‘nominated energy saver’. The application is
       independent of TfNSW

•   Reference Model
    Certificate issue requires the development of a baseline energy reference   [See Next Slide]
    model for each tranche
    • Data from each set in the tranche post modification needs to be
       collected, and energy savings determined over both Summer and Winter
    • Normalisation for passenger loads is permitted

•   Pilot Set
    For the NSW Energy Savings Scheme, the planned pilot set does not provide
    any useful results or contribution to the verification method.
