# Shipley Bidder Comparison Matrix

## Purpose

Systematic competitive assessment tool for comparing your position against competitors from the customer's perspective. The matrix follows the Shipley flow: Issues → Strengths/Weaknesses → Strategies → Action Plans → Features/Benefits → Themes.

## Two Assessment Methods

### Method 1: Weighted Scoring (Recommended for CBS)

Score each bidder 1-10 on each issue, then multiply by the issue weight.

| Issues | Weight | Us (Points) | Us (Score) | Competitor A (Points) | Competitor A (Score) |
|---|---|---|---|---|---|
| [Client hot button 1] | [1-10] | [1-10] | Auto | [1-10] | Auto |
| [Client hot button 2] | [1-10] | [1-10] | Auto | [1-10] | Auto |
| **Totals** | | | **Sum** | | **Sum** |

Highest total score = strongest position. Green = leader.

**Scoring guidance:**
- Base assessment on your best estimate of the **customer's perceptions**, not your internal view
- Early in sales cycle: use 1-3 weights (low/medium/high)
- After RFP release with evaluation criteria: use exact percentage weights in decimal form
- Apply consistent scoring standards across all bidders

### Method 2: Points Allocation (When evaluation criteria have defined points)

Allocate available points to each bidder per issue. Each bidder can independently receive up to the maximum available points.

| Issues | Available Points | Us | Competitor A | Competitor B |
|---|---|---|---|---|
| [Requirement 1] | [Max points] | [Assigned] | [Assigned] | [Assigned] |
| **Totals** | | **Sum** | **Sum** | **Sum** |

**Points guidance:**
- 10 points for basic compliance with a requirement
- Fewer if a bidder cannot meet the requirement
- More if the bidder exceeds the requirement
- Available points are NOT divided between competitors — each can receive the maximum

## Application to CBS Tender Scorecard

The Shipley Bidder Comparison Matrix maps to the CBS tender-scorecard skill:

| Shipley Issue | CBS Scorecard Dimension | Weight |
|---|---|---|
| Client hot buttons / requirements | Sector Alignment | 25% |
| Solution differentiation | CAPITAL Applicability | 20% |
| Price competitiveness | Contract Value | 15% |
| Incumbent/relationship strength | Client Relationship History | 15% |
| Competitor strengths vs ours | Competitive Position | 10% |
| Resource depth | Resource Availability | 10% |
| Strategic fit | Strategic Value | 5% |

## Template Usage

For each Go tender, the Research CBS agent should complete a Bidder Comparison Matrix as part of the Bronze phase competitor assessment. Use the weighted method with evaluation criteria percentages as weights.
