### Slide 2
Context
The NSW Energy Savings Scheme (ESS) has the potential to deliver funding back to Downer based on software and hardware upgrades to the Sydney train fleets.
It requires application of a framework, developed for fixed infrastructure, to mobile rolling stock, a new application of the ESS.
The application requires:
an approach that successfully models an increased number of complex variables 
satisfying ESS auditors and successfully managing the disposition of certificates on a live ESS trading platform
To take advantage of the software upgrade, CBS requires a commencement date of no later than 30 Nov 2020 to be certain SGT software modifications are compliant for ESCs.
2.  CBS has:
identified a path to manage these factors, and retained a qualified ESC (Energy Saving Certificates) issuer as a partner
We have proposed a value share model whereby Downer provides the data and pays CBS Group a recoupable retainer through project development
In response to Sean McCormack's request we have also costed a non-shared risk, T+M engagement
Shared risk/shared value is our preferred option.
It demonstrably delivers less risk and more value to Downer
However this requires executive sponsorship – hence our request for your review today to determine Downer's appetite for this proposition.

### Slide 7
Program Description
CBS’ ESC Partner, Certificate Issuer and Auditors have confirmed:

ApproachThe best approach is a ‘Measurement and Verification’ method treating the fleets as a series of tranches, defined by the number of sets that may be modified in each window prior to Winter or Summer
Brings forward the ESC payments (early as possible)
Allows for software and hardware modifications to be ‘assessed’ as tranches
Downer are the  ‘nominated energy saver’.  The application is independent of TfNSW

Reference ModelCertificate issue requires the development of a baseline energy reference model for each tranche
Data from each set in the tranche post modification needs to be collected, and energy savings determined over both Summer and Winter
Normalisation for passenger loads is permitted

Pilot SetFor the NSW Energy Savings Scheme, the planned pilot set does not provide any useful results or contribution to the verification method.
