## HVAC Audit Report 13 August 2024 - JD.docx

*File: `HVAC Audit Report 13 August 2024 - JD.docx`*

Audit Compliance Report: WTP Sets HVAC S/W Review



| Date: | 12th August 2024 |

| Required WTP HVAC S/W: | Version (1.20) |

| Number of Sets viewed physically: | 4 sets were viewed physically on 30 July 2024 |

| Number of sets viewed on the S/W Data Sheet: | 78 sets viewed |

|  |  |

| Discrepancies: | Discrepancies: |

| Non-compliance to the latest S/W: | 5 WTP sets have been identified as running an older software version (1.11B4) compared to the required ver(1.20) |

| Blank Tabs on the S/W Sheet: | 7 WTP sets did not display HVAC software version ver(1.20) on the report. |



| Root Cause Investigation: | Root Cause Investigation: |

| Non-compliance to the latest S/W: | Upon investigation of the issue, it was identified that the 5 WTP sets not running the correct software version were the ones that had their HVAC units replaced. The original units were installed with the correct HVAC s/w ver(1.20), but following a failure or during the maintenance overhaul, they were swapped out with units from our OEM's spares pool.

The root cause of the incorrect software upload has been identified as Human error. The software update for all 16 HVAC units on the train is applied at the train set level, meaning the s/w is installed simultaneously across all HVAC units in the set rather than individually when a unit is replaced due to a defect. It has been identified that there may have been assumptions that HVAC units returning from the OEM have s/w ver(1.20) pre-installed and do not require an update.

Downer's current process dictates when HVAC units are replaced as part of an overhaul / Defect, the software is verified according to the document ‘08-202 - HVAC Replacement procedure, which is a part of our TMP - CEP000333. |

| Blank Tab on the S/W Sheet: | The investigation determined that the root cause was blank tabs resulting from missing information extracted from the Train Information System. This issue is classified as a Train Communication reporting error and does not indicate any software discrepancy or failure of the HVAC unit.

The problem arose due to a lack of response from the Ethernet switch, which connects the HVAC to the Train Information System. Consequently, software information from the HVAC and other equipment was not properly reported properly. However, this only led to an information loss on the network and did not affect the functionality or performance of the systems. |



| Corrective Actions: | Corrective Actions: | Preventive Actions: |

| Non-Compliance to the latest S/W: | Work Orders were raised on August 7th, 2024, to install the latest HVAC s/w ver(1.20) on the 5 WTP Sets. The installations were completed on August 9th, 2024. As a result, all 78 WTP Train sets have now been upgraded to the latest HVAC software version. (Closed Work order pdf’s attached) | In accordance with Downer Standard DG-QA-ST005 – Business Improvement Standard, Downer will implement the following tasks to ensure the current software version is maintained on the train sets:.

Software Management: Moving forward, software updates will be managed at the asset level through our Asset Management System (Maximo), rather than at the set level. This change will enhance governance and mitigate risks associated with discrepancies in HVAC software versions.

Maximo Model Update: The Systems Engineering Department at Downer is currently updating the Maximo (Downer Rail’s Asset Management System) model to accept only the latest software version. In the event of a HVAC unit is installed onto a set with the incorrect S/W a build discrepancy will be flagged, triggering an intervention action.

New Procedures: New job plans will be created to address any build discrepancies flagged by the system.

Daily Compliance Monitoring: Software reports will be generated and monitored daily to verify compliance against the software revisions on all train sets. Any discrepancies detected will prompt an intervention action. |

| Blank Tabs on the S/W Sheet: | The Downer Team has manually inspected the Train Sets by reviewing the eTIS screens and connecting laptops to verify the HVAC software version.  This inspection confirmed that the latest version (1.20) has been installed on the train sets.

After the manual inspection by the team, the daily reports were cross-checked and it was confirmed that previous blank tabs are now auto populated, displaying the latest s/w ver(1.20). | As per Downer Standard: DG-QA-ST005 – Business Improvement Standard, Downer will implement the following tasks to ensure the current version of s/w is installed on the train sets.

Intervention Trigger: If a blank tab is identified during the daily software report review, an intervention action will be initiated by the Downer Engineering Team.

Intervention Steps:
Gain access to the train set 
Conduct a manual inspection to verify that the latest version of the HVAC software installed.
Repair or replace any defective assets as necessary |
