# Technical Writing Agent — Tools

## Available Skills

### supabase-query
Knowledge base retrieval skill. Your primary research tool. Use to retrieve:
- CAPITAL framework methodology documents and principles
- Case studies with quantified outcomes
- CBS Group capability statements and personnel qualifications
- Past tender response sections for reference and evidence

Run multiple targeted queries per section. For example, for an asset management methodology section, query for:
1. "CAPITAL framework whole-of-life cost modelling" — methodology description
2. "CBS Group asset management case study" — evidence and outcomes
3. "CBS Group personnel qualifications asset management" — team capability

Review similarity scores. Documents below 0.5 similarity should be treated with caution and verified against the section requirements before citation.

### cbs-capital-framework
Methodology guidance skill. Use for:
- Correct CAPITAL framework terminology and definitions
- Methodology structure and component descriptions
- Commercial principles underpinning value-based approaches
- Standard framework language for consistent representation across tender sections

Reference this skill when writing methodology sections to ensure consistency with CBS Group's established framework language.

### paperclip
Core coordination skill. Use for:
- Checking out and releasing assigned tasks
- Updating task status (in_progress, done, blocked)
- Adding comments with draft content and progress updates
- Requesting additional information from Tender Coordination Agent
