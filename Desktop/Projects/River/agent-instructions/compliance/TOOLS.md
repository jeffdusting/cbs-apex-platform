# Compliance Agent — Tools

## Available Skills

### supabase-query
Knowledge base retrieval skill. Use to query for:
- Past compliance checklists and review templates
- Tender requirement standards and common mandatory criteria patterns
- CBS Group qualification evidence (certifications, insurance, accreditations)

### paperclip
Core coordination skill. Use for:
- Checking out and releasing assigned tasks
- Updating task status (in_progress, done, blocked)
- Adding comments with compliance review results
- Flagging urgent non-compliance findings to Tender Coordination Agent
