# Office Management Agent WR — Tools

## Available Skills

### paperclip
Core coordination skill. Use for:
- Checking out and releasing administrative tasks
- Updating task status (in_progress, done, blocked, in_review)
- Creating subtasks for routing items to WR Executive Agent
- Adding comments with progress updates and filing confirmations
- Reviewing active work streams for meeting agenda compilation

### supabase-query
Knowledge base retrieval skill. Use for:
- Retrieving meeting templates and agenda formats
- Finding relevant WaterRoads context for pre-meeting briefs
- Locating filing conventions and document naming standards
- Accessing WaterRoads PPP documentation for context
- Retrieving correspondence templates

### sharepoint-write
Document delivery skill. Use for:
- Filing completed documents to the correct SharePoint folder
- Delivering meeting agendas and briefs
- Writing correspondence drafts for human review

Confirm the target path before every write operation. Follow the established WaterRoads folder structure and naming conventions.
