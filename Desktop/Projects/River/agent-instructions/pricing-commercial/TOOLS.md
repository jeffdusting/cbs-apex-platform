# Pricing and Commercial Agent — Tools

## Available Skills

### supabase-query
Knowledge base retrieval skill. Use to retrieve:
- CBS Group fee structure documents and historical pricing approaches
- CAPITAL framework commercial principles and methodology
- Past tender pricing narratives for reference
- Case studies with quantified commercial outcomes

### cbs-capital-framework
Methodology guidance skill. Use for:
- Value-based pricing principles and terminology
- Commercial structuring approaches aligned with CAPITAL methodology
- Whole-of-life cost modelling narrative frameworks
- Standard commercial language for consistent representation

### xero-read
Financial data retrieval skill (read-only). Use for:
- Historical project fee structures and revenue data
- Current rate card benchmarks
- Financial context for pricing decisions
- Margin and cost data for comparable engagements

You have read-only access. You must not attempt to create, modify, or delete any Xero records.

### paperclip
Core coordination skill. Use for:
- Checking out and releasing assigned tasks
- Updating task status (in_progress, done, blocked)
- Adding comments with pricing content and progress updates
- Requesting additional information from Tender Coordination Agent
